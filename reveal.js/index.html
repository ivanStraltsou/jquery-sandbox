What is jQuery?

jQuery is a fast, small, and feature-rich JavaScript library. It makes things like HTML document traversal and manipulation, event handling, animation, and Ajax much simpler with an easy-to-use API that works across a multitude of browsers. With a combination of versatility and extensibility, jQuery has changed the way that millions of people write JavaScript.

Basic CSS selectors

API reference 

  // All selector
  $('*');

  // Class selector
  $('.class-selector');

  // Element selector
  $('p');

  // ID selector
  $('#idSelector');

  // Multiple selector
  $('.class-selector, #idSelector, p');

Attribute selectors

API reference 

  // attribute, with any value
  $('[name]');

  // attribute with a value exactly equal to a certain value
  $('[name="value"]');
  // Matches elements that match all of the specified attribute filters
  $('[name="value"][name2="value2"]');

  // attribute with a value either equal to a given string or starting with that string followed by a hyphen (-)
  $('[name|="value"]');
  // attribute with a value containing the a given substring
  $('[name*="value"]');
  // attribute with a value containing a given word, delimited by spaces
  $('[name~="value"]');
  // attribute with a value ending exactly with a given string. The comparison is case sensitive
  $('[name$="value"]');
  // don’t have the specified attribute, or do have the specified attribute but not with a certain value
  $('[name!="value"]');
  // attribute with a value beginning exactly with a given string
  $('[name^="value"]');

Hierarchy

API reference 

  // Child Selector
  $('.list > li');

  // Descendant Selector (“ancestor descendant”)
  $('.list li');

  // Next Adjacent Selector (“prev + next”)
  $('.list li:first-child + li');

  // Next Siblings Selector (“prev ~ siblings”)
  $('.list li.specific ~ li');

Form

API reference 

  // Selects all button elements and elements of type button
  $(':button');

  // Selects all elements of type checkbox
  $(':checkbox');

  // Matches all elements that are checked or selected
  $(':checked');

  // Selects all elements that are disabled
  $(':disabled');

  // Selects all elements that are enabled
  $(':enabled');

  // Selects all elements of type file
  $(':file');

  // Selects element if it is currently focused
  $(':focus');

  // Selects all elements of type image
  $(':image');

  // Selects all input, textarea, select and button elements
  $(':input');

  // Selects all elements of type password
  $(':password');

  // Selects all elements of type radio
  $(':radio');

  // Selects all elements of type reset
  $(':reset');

  // Selects all elements that are selected
  $(':selected');

  // Selects all elements of type submit
  $(':submit');

  // Selects all elements of type text
  $(':text');

Optimizations


// Bad:
$('form.person input:checkbox');

// Better:
$('form.person input').filter(':checkbox');

// Best:
$('input[type=“checkbox”]', 'form.person');
